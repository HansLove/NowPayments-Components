## Dark Mode Support

### Automatic Dark Mode
```css
:root {
  /* Light mode (default) */
  --np-surface: #ffffff;
  --np-on-surface: #1f2937;
}

@media (prefers-color-scheme: dark) {
  :root {
    --np-surface: #1f2937;
    --np-surface-variant: #374151;
    --np-on-surface: #f9fafb;
    --np-on-surface-variant: #d1d5db;
    --np-border: #374151;
  }
}
```

### Manual Dark Mode Toggle
```css
/* Light theme */
[data-theme="light"] {
  --np-surface: #ffffff;
  --np-on-surface: #1f2937;
  --np-border: #e5e7eb;
}

/* Dark theme */
[data-theme="dark"] {
  --np-surface: #1f2937;
  --np-surface-variant: #374151;
  --np-on-surface: #f9fafb;
  --np-border: #374151;
}
```

## Animation Variables

```css
:root {
  /* Transition durations */
  --np-transition-fast: 0.15s;
  --np-transition-base: 0.2s;
  --np-transition-slow: 0.3s;

  /* Easing functions */
  --np-ease-out: cubic-bezier(0, 0, 0.2, 1);
  --np-ease-in: cubic-bezier(0.4, 0, 1, 1);
  --np-ease-in-out: cubic-bezier(0.4, 0, 0.2, 1);
}

/* Apply to components */
.np-button,
.np-input,
.np-modal {
  transition-duration: var(--np-transition-base);
  transition-timing-function: var(--np-ease-out);
}
```

## Usage Example

**Important**: Always import the base styles first, then your custom overrides:

```tsx
import { NowPaymentsProvider, DepositModal } from '@taloon/nowpayments-components'
import '@taloon/nowpayments-components/styles'  // ← Base styles first
import './my-custom-theme.css'                   // ← Your overrides second

function App() {
  return (
    <div className="caos-theme"> {/* Apply scoped theme */}
      <NowPaymentsProvider apiKey="your-api-key">
        <DepositModal {...props} />
      </NowPaymentsProvider>
    </div>
  )
}
```

### CSS Import Order Matters

```tsx
// ✅ Correct order
import '@taloon/nowpayments-components/styles'  // Base styles
import './my-theme-variables.css'              // Custom variables
import './my-component-overrides.css'          // Specific overrides

// ❌ Wrong order - your styles may be overridden
import './my-custom-styles.css'
import '@taloon/nowpayments-components/styles' // This will override your styles
```
